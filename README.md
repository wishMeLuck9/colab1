# Fish Dishes Clustering Analysis

This repository contains a Google Colab / Jupyter Notebook project focused on the segmentation of fish dishes using clustering methods.

The main goal of the project is to group fish dishes into meaningful clusters based on numerical features from the dataset `dania.rybne.csv`.

## Project Goal

The purpose of this analysis is to compare two clustering approaches:

- hierarchical clustering using Ward linkage;
- k-means clustering.

The project evaluates different numbers of clusters and compares the results using clustering quality metrics.
